SharePoint Document Library web part
The SharePoint Document library web part allows us to select a single document library inside a SharePoint site page.
Users with proper permissions can edit, view, and add files straight from the document library web part itself.
By default, it will choose the document library name as a title for the document library web part. If we want to change the title, we can edit the document library web part and change the title in the title area.
Users can able to view, filter, sort, and also pin a document library, by selecting the See all options users can view the document library.

How to add Document Library web part in SharePoint online

Open the SharePoint site to add document library web parts to the SharePoint modern site page.
Select Edit Page in the modern SharePoint site page.
Add a new Section layout by clicking on the + circled icon.
In the search box, enter the document library and select a document library web part.
After adding the Document Library Web Part, it automatically displays the document libraries that are available on the current SharePoint site
Select a document library directly or from a drop-down, which you want to display on your SharePoint site page. Click on the Apply button.
